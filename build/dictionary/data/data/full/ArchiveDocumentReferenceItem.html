ArchiveDocumentReferenceItem (pc_archivedocrefitem) (delegates to Editable)
Description
Keep track of one bean in database that is directly referenced by one bean from an archive document. One bean in database can be referenced by more than one archive document, in which case there are multiple ArchiveDocumentReferenceItem rows for it.
Entity: ArchiveDocumentReferenceItem
Attributes: Editable?, Final?, Keyed?, Sourceable?, Versionable?
