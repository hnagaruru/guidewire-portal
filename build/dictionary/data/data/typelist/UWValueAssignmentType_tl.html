UWValueAssignmentType     (pctl_uwvalueassignmenttype)
Specifies the way in which a default value is assigned to the "Reference Value"
Attributes:

Referenced by:
UWIssueType.DefaultValueAssignmentType
Typecodes
Code Name Description Priority Internal Retired
Fixed Fixed Indicates that the default value for the approval is copied directly from the issue   true  
OffsetAmount Offset Amount A fixed amount is added to or subtracted from the issue's reference value to compute a default for the approval   true  
OffsetPercent Offset Percentage The issue's reference value is increased or decreased by a percentage amount, depending on the direction of the comparator   true
